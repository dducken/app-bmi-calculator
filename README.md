### Calculador de Indice de Masa Corporal (BMI) utilizando principalmente Angular 🤩

### ✨ Función principal:
* #### **`Calcular el BMI dependiendo la altura, peso y edad de la persona`** 

### 🖐️ Sirve para:
* #### **`Conocer si tenes un peso bajo, normal o sobrepeso`**
